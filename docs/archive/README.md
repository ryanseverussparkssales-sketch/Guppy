# Archive Index

This folder holds historical docs that are no longer part of the living project-status surface.

## Active Docs

- `README.md` at repo root — stable project source of truth
- `ROADMAP.md` at repo root — active priorities and handoff log

## Archive Buckets

### `root-history/`

Former repo-root session, milestone, review, and one-off briefing docs moved out of the root to reduce noise.

### `planning-history/`

Older planning boards, audits, checklists, and handoff/reference docs moved out of `docs/` once they stopped being part of the active working set.

### `reference-history/`

Superseded broad reference docs moved out once their useful active summary was folded into the living docs.

## Guidance

- Add new implementation notes to `ROADMAP.md`, not here.
- Keep active operational references in `docs/` only if they are still used regularly.
- Move superseded planning/status docs into the appropriate archive bucket instead of leaving them beside living docs.
